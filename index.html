Project on Console

Visual Studio Code is an IDE coding program which allows the user to code offline. Visual Studio Code can be used by creating a folder and opening that folder onto Visual Studio Code.

The console screen is used for testing and seeing if the code is working properly or if its correct. Printing onto a console can be done by writing "console.log(item you want to put in)". To open the console screen, you have to right click, or pressing ctrl or command+shift+c and pressing console.
